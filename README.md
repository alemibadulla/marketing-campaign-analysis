# marketing-campaign-analysis
📊 About
End-to-end analysis of a marketing campaign dataset: from data cleaning to prescriptive recommendations on target audience.
🎯 Goal
Identify the key drivers of customer response to a marketing campaign and build a profile of the "ideal customer".
📁 Data

Source: Kaggle
~2,200 customers
Features: income, education, marital status, spending by category, purchase channels

🛠️ Stack
Python Pandas Matplotlib Seaborn
🔍 What was done

Data cleaning: missing value imputation (median), duplicate removal
Descriptive, diagnostic, predictive, and prescriptive analysis
Customer segmentation by income, education, and marital status
Correlation analysis and boxplot visualizations
Monthly spending trend analysis using rolling average
Final dashboard with 4 charts

📌 Key Findings

Customers with high income showed the strongest campaign response (correlation Response–MntTotal = 0.66)
August is the peak spending month — optimal time to launch campaigns
Offline store showed the highest correlation with total spending (0.67)
